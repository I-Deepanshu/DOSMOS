# DOSMOS: The Premium Identity Universe

## 🌌 Project Overview
DOSMOS is a production-grade identity authentication and admin console platform designed with a "Cinematic Industrial" aesthetic. It transforms traditional administrative dashboards into a living solar system where every user identity is represented as a unique planet within a shared cosmos.

---

## 🛰️ Senior Elite UX Physics
We have implemented a high-end interaction engine that prioritizes "Natural Intelligence" over mechanical transitions:

### 1. Intent Detection Engine
- **Velocity Tracking**: The system calculates cursor velocity in real-time.
- **Result**: It distinguishes between accidental swipes and intentional targeting. Orbits only stabilize when the pilot decelerates near a planet.

### 2. Cinematic Motion Hierarchy
- **Partial Slowdown**: On hover, the targeted planet locks in place while the rest of the solar system enters a **0.2x Cinematic Slowdown**.
- **The Energy Shift**: Clicking an identity triggers a focused "Focus Lock" sequence—1.12x scale expansion, interior Rim Lighting, and a background dimming effect.
- **Atmospheric Resumption**: Motion resumes based on behavior. Short hovers result in instant restarts, while deep focuses trigger a "Soft Startup" (80ms buffer).

### 3. Universal Pilot Experience
- **Rocket Cursor**: A custom-designed SVG Rocket is now the universal navigator across all pages, turning every interaction into a piloting experience.
- **Upscaled Presence**: Planetary volumes were increased to **20px–32px** to ensure every identity has physical substance in the viewport.

## 🛡️ Authentication & Integrity
The core security layer is built for high-trust environments:

- **Multi-Step Verification**: A seamless DOB and Security Question flow that acknowledges the "Cosmic Origin" of the user.
- **Social Population**: Backend middleware performs deep lookups of identity profiles, ensuring names and planetary contexts are always resolved (preventing "Unknown" identities).
- **Planet Birth Sequence**: A custom registration flow that procedurally generates a unique planet type (Rocky, Ocean, Gas Giant, Ice) and color palette for every new user based on their cosmic signature (DOB).

## 💻 Technical Architecture
- **Frontend**: Next.js 16/React with Framer Motion and custom CSS Keyframe Orchestration.
- **Backend**: Node.js & Express with MongoDB (Mongoose).
- **Communication**: real-time Socket.io for live data channels and chat.
- **Security**: JWT-based session management with Refresh Token rotation and social middleware.
